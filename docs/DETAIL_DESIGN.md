# 안전보건 점검관리 웹앱 상세설계서 v1

## 1. 구축 목표
본사 안전보건팀 3명이 사용하는 정기안전보건평가 및 중대재해처벌법 준수평가 반자동화 시스템을 구축한다.

## 2. 1차 기술 구성
- Next.js
- Vercel
- Google Drive
- Google Sheets
- PPTX/XLSX/PDF 생성 모듈
- AI 사진 분석 및 지적사항 추천

## 3. 권한
- 본사 안전보건팀: 로그인 필요
- 현장 안전관리자: 로그인 없이 조치 링크 접속

## 4. Drive 폴더
- 01_템플릿
- 02_점검결과
- 03_조치사진
- 04_최종본

## 5. Sheets DB
현장목록, 점검목록, 지적사항, 조치관리, 사용자, 템플릿설정 시트 사용.

## 6. 템플릿 보호 원칙
PPTX 표 크기, 글꼴, 글꼴 크기, 위치, 인쇄영역은 변경하지 않는다. 텍스트가 길면 AI 요약 후 사용자가 수정한다. 폰트 크기는 자동 축소하지 않는다.
